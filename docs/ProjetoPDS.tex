% truncar a resposta impulsiva ideal (infinita) h_ideal[n] = fc/fs * sinc(fc/fs * n)
% multiplicando-a por uma janela temporal w[n] de comprimento finito N:
%   h[n] = h_ideal[n] * w[n],   n = 0, 1, ..., N-1
% A escolha da janela determina o compromisso entre largura da banda de transição
% e atenuação na banda de rejeição (stop-band):
%   - Hamming: atenuação ≈ 41 dB, ondulação passband ≈ 0.0194 dB
%   - Blackman: atenuação ≈ 74 dB, banda de transição mais larga que Hamming
%   - Kaiser (β=8.6): atenuação ≈ 60 dB, β ajustável — melhor controle do trade-off
%
% Hint: Destaque que filtros FIR de fase linear (Tipo I, N ímpar) introduzem
% atraso de grupo constante de (N-1)/2 amostras. Para N=32 taps e fs=7680 Hz,
% o atraso é 15/7680 ≈ 2,0 ms. Para N=128: 63/7680 ≈ 8,2 ms. Este atraso é
% irrelevante em análise offline mas crítico em relés de proteção em tempo real.
